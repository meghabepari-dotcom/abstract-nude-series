# Abstract Nude Illustration Series

## 🎨 Overview
An interactive gallery showcasing 24 illustrations. The series is built on **4 core designs**, with each design rendered in **6 unique styles**. This project combines artistic series work with a clean, logical user interface.

## 🧠 The Concept
As a Business Analyst with a background in reporting and analytics, I designed this experience to be both a creative showcase and a functional application. It features:

* **Custom Match Logic:** A pairing system that allows users to find connections between different styles of the same design.
* **User Flexibility:** A "Skip to Gallery" option for immediate access to the full collection.
* **High-Contrast UI:** A dark-mode gallery aesthetic designed to make the illustrations stand out.

## 🕹️ How to Use
1.  **Interact:** Match two tiles from the same design series.
2.  **Explore:** Once the tiles are matched (or skipped), the full static gallery is revealed.
3.  **Connect:** Reach out via the integrated LinkedIn link to discuss the work.
