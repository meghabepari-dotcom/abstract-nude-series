# Abstract Nude Illustration Series

## 🎨 Overview
An interactive gallery experience developed to showcase a series of **24 illustrations by artist Brishti Bepari**. 

The project features **4 core designs**, each rendered in **6 unique styles**. I designed and developed this interface to transform a static art series into a logical, interactive UX case study.

## 🧠 The Concept
As a Business Analyst, I built this application to demonstrate how technical logic can enhance creative storytelling.
