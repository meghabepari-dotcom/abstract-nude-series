# Abstract Nude Illustration Series

## 🎨 Overview
An interactive gallery experience developed to showcase a series of **24 illustrations by artist Brishti Bepari**. 

The project features **4 core designs**, each rendered in **6 unique styles**. I designed and developed this interface to transform a static art series into a logical, interactive UX case study.

## 🧠 The Concept
As a Business Analyst with a background in reporting and analytics, I built this application to demonstrate how technical logic can enhance creative storytelling. Highlights include:

* **Custom Match Logic:** A pairing system that allows users to find connections between different styles of the same design.
* **User Flexibility:** A "Skip to Gallery" option for immediate access to the full collection.
* **High-Contrast UI:** A dark-mode gallery aesthetic designed to make the illustrations stand out.

## 🕹️ How to Use
1.  **Interact:** Match two tiles from the same design series.
2.  **Explore:** Once the tiles are matched (or skipped), the full static gallery is revealed.
3.  **Connect:** Use the links below to reach out to the artist or the developer.
